About this Stream

This page displays a stereo camera feed from a ZED camera. The stereo feed contains side-by-side left and right camera views for VR/AR applications.

Troubleshooting:

  • This page must be accessed using http://, not https://
  • The WebRTC server must be running on port 8080
  • If you see a black screen, try clicking "Disconnect" then "Connect" again
  • Make sure your device and the server are on the same network
